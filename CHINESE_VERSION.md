# Obsidian Git 插件 - 中文使用指南

## 🎉 简介

这是 Obsidian Git 插件的**完全汉化版本**，提供了完整的简体中文界面支持。

## ✨ 主要特性

- 🌏 **完整的国际化支持** - 插件核心功能已全部翻译为简体中文
- 🔄 **自动语言检测** - 根据系统语言自动选择界面语言
- ⚙️ **灵活的语言设置** - 可在设置中手动切换中英文
- 🚀 **无缝体验** - 所有命令、通知、设置均已翻译

## 📦 安装方法

### 方式一：手动安装

1. 下载或克隆此仓库
2. 将项目文件夹复制到 Obsidian 插件目录：`<vault>/.obsidian/plugins/`
3. 在项目目录运行：
   ```bash
   npm install
   npm run build
   ```
4. 在 Obsidian 中启用插件

### 方式二：从源代码构建

```bash
# 安装依赖
pnpm install

# 开发模式（自动重新构建）
pnpm run dev

# 生产构建
pnpm run build
```

## 🌐 语言设置

### 自动检测

插件默认会根据您的系统语言自动选择界面语言：
- 简体中文系统 → 显示简体中文界面
- 其他语言系统 → 显示英文界面

### 手动设置

1. 打开 Obsidian 设置
2. 选择 "Obsidian Git" 插件设置
3. 在页面顶部的"语言 / Language"选项中选择：
   - **自动（跟随系统）** - 根据系统语言自动选择
   - **English** - 英文界面
   - **简体中文** - 中文界面

4. 建议重新启动 Obsidian 以确保所有界面完全更新

## 🔧 使用示例

### 基本工作流程

1. **初始化仓库**
   - 命令：`Git: 初始化仓库` 或 `Git: 克隆仓库`

2. **提交更改**
   - 命令：`Git: 提交并推送` （快捷键可在设置中配置）
   - 或在源代码管理视图中手动暂存和提交

3. **同步文件**
   - 自动：在设置中配置自动提交和推送间隔
   - 手动：使用 `Git: 拉取` 和 `Git: 推送` 命令

4. **查看更改**
   - 命令：`Git: 打开源代码管理视图`
   - 命令：`Git: 打开历史记录视图`

### 中文界面示例

```
命令面板中的命令：
- Git: 提交
- Git: 提交并推送
- Git: 拉取

- Git: 暂存当前文件
- Git: 打开源代码管理视图
- Git: 打开历史记录视图
```

```
通知消息示例：
- "已提交 5 个文件"

```
